## 🗺️ Projekt 1: WuppertalMapMvc
Interaktive Karte der Bushaltestellen in Wuppertal mit ASP.NET Core MVC und D3.js.  
**Technologien:** ASP.NET Core, D3.js, Entity Framework Core, Bootstrap  

### Funktionen
- CRUD für Haltestellen  
- Visualisierung per D3.js  
- API `/api/stop`  
- Datenbank mit Seed-Daten  

### Start
1. Öffne `WuppertalMapMvc.sln` in Visual Studio 2022  
2. `dotnet ef database update`  
3. `F5` oder `Strg + F5`

---

## 📚 Projekt 2: LernpfadApp
Webplattform zur Erstellung interaktiver Lernpfade mit CKEditor und Datei-Uploads.  
**Technologien:** ASP.NET Core MVC, Entity Framework Core, CKEditor 5, Bootstrap 5  

### Funktionen
- Lernpfade mit Titel, Beschreibung, Vorschau-Bild  
- CRUD-Operationen  
- Rich-Text-Editor (CKEditor)  
- Datei-Upload in `/uploads`  

### Start
1. Öffne `LernpfadApp.sln` in Visual Studio 2022  
2. `dotnet ef database update`  
3. `F5` oder `Strg + F5`
